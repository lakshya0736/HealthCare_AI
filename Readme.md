# AI Healthcare Web App

An AI-powered healthcare web application designed to provide quick medical guidance for users with limited access to healthcare resources. The platform helps users analyze symptoms, get possible disease suggestions, receive home remedies, consult doctor advice, find nearby hospitals, and manage appointments.

###Features

- **AI Symptom Analysis**
  - Users can enter symptoms and get AI-based health guidance.
  - Provides possible disease suggestions based on symptoms.

- **Disease Suggestions & Health Advice**
  - Gives basic information about possible health conditions.
  - Provides precautionary measures and doctor recommendations.

- **Home Remedies**
  - Suggests general home remedies for common health issues.

- **Nearby Hospital Lookup**
  - Helps users locate nearby hospitals for medical assistance.

- **Emergency Support**
  - Provides quick access to emergency contact/call support.

- **Appointment Booking**
  - Users can book doctor appointments.
  - Appointment details are stored securely in the backend.

-**Backend Data Management**
  - Handles user requests and stores appointment information.
